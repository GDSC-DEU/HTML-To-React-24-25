자바스크립트 기초

onclick 속성

html 속성으로 속성 값에 해당하는 자바스크립트 코드를 해당 요소가 클릭 되었을 때(클릭 이벤트 발생) 실행!

예시 

            < input type="button" value = "dark" onclick ="
                // 여기서 부터 자바스크립트
                /*
                    버튼을 클릭하면, 버튼 이벤트가 발생하고
                    현재 문서 html 태그 중에서 css 선택자가 'body' 즉 body 태그를 선택하고
                    body 태그의 스타일 속성 중 배경색 속성 값을 'black'으로 바꿔서 배경색을 바꾸어 주는 코드
                */
                document.querySelector('body').style.backgroundColor = 'black';
            ">
실행 해보기 : black
실행 해보기 : white

실행 해보기 : 다크모드
